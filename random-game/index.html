Master Mind is a logic game where computer generates an n-position code of several pre-determined colors and player should guess the code. The code can consist of all colors or just a single color of the palette.

To guess the code player is given 7 attempts. After an attempt is complete, computer shows how many attempt positions have the same color and place as the code and how many remaining positions have the same color but different place.

Player has 700 points of score in the beginning of the game and every attempt reduce the score by 100 points.

After player wins the game he is allowed to go forward to the next level.

Master Mind - это логическая игра в которой компьютер генерирует код, состоящий из нескольких заранее определенных цветов. В коде может использоваться только часть цветов или даже один цвет из палитры.

Игрок должен отгадать код за 7 попыток. После каждой попытки компьютер показывает сколько позиций совпало по расположению и цвету, и сколько из оставшихся позиций имело нужный цвет но распологалось в другом месте.

В начале раунда игроку дается 700 баллов и за каждую попытку эта цифра уменьшается на 100.

После того как игрок отгадал код, он может перейти на следующий уровень.
